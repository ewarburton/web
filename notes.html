[c:Hz] sets base frequency (261 by default, e.g., [c:440] for A4) [m:X/Y] plays just intonation interval (e.g., [m:5/4] for major third) [t:X] plays equal temperament interval in semitones (e.g., [t:7] for perfect fifth) [t:X\Y] plays interval in EDO steps (e.g., [t:7\12] for perfect fifth in 12EDO) [!] mutes the base note (e.g., [m:5/4,!] plays major third only) ,key=K binds to keyboard key (e.g., [t:7,key=f] plays when 'f' is pressed) ,rgb:R,G,B can be added to any code to set button color (e.g., [m:5/4,rgb:255,100,100])
